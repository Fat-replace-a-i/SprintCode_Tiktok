## Exécuter localement

Cloner le projet

```bash
  git clone https://link-to-project
```

Aller dans le répertoire du projet

```bash
  cd Usergen
```

Installer les dépendances

```bash
  npm install
```

Démarrer le serveur

```bash
  npm run dev
```
